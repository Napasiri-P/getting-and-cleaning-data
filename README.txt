The modified data set includes:
=========================================

- "README.txt"

- "features.txt" : list of all features presented as column names

- "features_info.txt" : show information of each feature

- "run_analysis.R" : R script showing steps from original data to modified data set

- "tidy_data_set/data_mean_std.csv" : data set with only features that are mean or standard deviation

- "tidy_data_set/data_mean_summary.csv" : average of each feature grouped by subject and activity


Notes: 
======
- Features are normalized and bounded within [-1,1].
- Each feature vector is a row on the text file.

original data and further information can be found on
=========================================
